# github-io-relay

Static relay page for GitHub Pages.

## Host

Upload `index.html` into your GitHub Pages repo (for example under `/vpn-bridge/`).

Example URL:

`https://your-user.github.io/vpn-bridge/index.html`

## Configure before deploy

Edit `index.html` and set these constants:

- `IRAN_WS_BASE`
- `NODE_WS_BASE`

Do not pass endpoint URLs from query params anymore.

## Behavior

- If required query params are missing, page stays empty.
- Required params:
  - `uuid`
  - `session` (short opaque string generated by iran-bridge tab launcher)
- Opens both sockets and forwards raw binary/text frames both directions.
- Reconnects automatically on disconnect.

## Example generated URL (opened by iran-bridge tab)

`https://your-user.github.io/vpn-bridge/index.html?uuid=f398641f-0e38-466d-b785-375935486020&session=bridge-f398641f-0e38-466d-b785-375935486020`
